
# Vig-jr

An Ai based discord bot trained to be act like me ( Papavig aka Vignesh Esakkiappan )


## Run Locally

Clone the project

```bash
  git clone https://github.com/Papavig/Vig-jr.git
```

Go to the project directory

```bash
  cd Vig-jr
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node .
```
